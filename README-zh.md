### 简介

​	根据公司的实际开发需求，初步梳理出符合公司开发状态的前端和后端开发规范，以便于未来团队代码风格的统一。



### node版本

```text
16.16.0
```



### 安装依赖

```bash
npm install
```





### 运行项目

```bash
npm run dev
```



### 清除缓存

```bash
npm run clean
```



### 更新vuepress

```bash
npm run update-package
```
